= Modification Made on Top of the Official AWS SDK for Ruby

1. Added S3::S3Object::copy_from_versioned() to return ObjectVersion
   when versioning is enabled, or return the object if versioning is
   disabled.

= How to Use it

  In your Gemfile, include the following line:

    gem 'aws-sdk', '0.0.0', :git => 'git://github.com/stzlei/aws-sdk-for-ruby-modified.git'

= Official AWS SDK

  Offiical AWS SDK is at https://github.com/amazonwebservices/aws-sdk-for-ruby
